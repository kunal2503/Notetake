# 📝 Notes App

A full-stack Notes Application that allows users to securely create, edit, delete, and manage their personal notes with authentication.

---

## 🚀 Features

- 🔐 User Authentication (Login / Register)
- 📝 Create, Read, Update, Delete Notes (CRUD)
- 💾 Persistent Storage (Database)
- 🔎 Search & Filter Notes
- 🌙 Dark Mode (optional)
- 📱 Responsive UI
- 🔔 Toast Notifications

---

## 🛠️ Tech Stack

### Frontend
- React.js
- HTML5, CSS3, JavaScript
- Axios (API calls)

### Backend
- Node.js
- Express.js

### Database
- MongoDB (Mongoose)

### Authentication
- JWT (JSON Web Tokens)
- bcryptjs (Password Hashing)
